## 6. 联调中发现并修复的问题

1. **MySQL Flyway 迁移失败**
   - 现象：V2 `import_items` 唯一键 `(job_id, relative_path)` 在 `utf8mb4` 下超过 MySQL 3072 字节索引限制。
   - 修复：`relative_path` 从 `VARCHAR(1000)` 调整为 `VARCHAR(700)`，同步修改 JPA `ImportItem.relativePath` 长度。
   - 回归：新增 `MigrationCompatibilityTest`，验证索引字节预算不超过 3072。

2. **舆情页未展示比赛级风险**
   - 现象：`sentiment_risk_assessments.factor_id IS NULL` 的比赛级风险已入库，但前端只展示关联到当前因素的风险。
   - 修复：舆情页新增“比赛级风险评分”表，并让风险统计至少覆盖当前比赛全部风险。
   - 回归：新增 `sentiment-view.test.ts`，覆盖 `PUBLIC_OVERHEAT / MEDIUM / 热门过热` 展示。

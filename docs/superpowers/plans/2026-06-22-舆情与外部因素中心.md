# 舆情与外部因素中心 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建 A1 + A2 舆情与外部因素中心，保存事实记录、风险评分、时效提醒，并提供前后端查询页面。

**Architecture:** 使用 V6 迁移新增 `match_context_factors` 与 `sentiment_risk_assessments`；增强 SOURCE JSON 批准入库，把外部因素和舆情记录拆成 factors，把风险评分拆成 risks；新增 `sentimentcenter` 后端查询 API 与 `/sentiment` 前端页面。所有新增行为先写测试，遵守认证和“不输出下注建议”的边界。

**Tech Stack:** Java 17, Spring Boot 3.3.6, JdbcTemplate, Flyway, H2/MySQL, Vue 3, TypeScript, Element Plus, Vitest, OpenSpec。

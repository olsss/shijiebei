# 赛前分析作战室与数据完整性中心 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建赛前分析作战室与数据完整性中心，按单场比赛聚合已有正式业务数据并展示缺失、过期和冲突检查。

**Architecture:** 不新增数据库表；新增 `prematchworkbench` 后端只读查询服务和 `/prematch-workbench` 前端页面。完整性检查由后端基于现有表计算，前端只展示数据准备状态，不生成新分析或下注建议。所有代码行为先写失败测试，再实现最小代码通过。

**Tech Stack:** Java 17, Spring Boot 3.3.6, JdbcTemplate, H2/MySQL, Vue 3, TypeScript, Element Plus, Vitest, OpenSpec。

# 赔率中心与盘口快照 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建赔率中心与盘口快照模块，确保每个玩法下的所有选项赔率都结构化入库并可在前后端查看。

**Architecture:** 使用 V5 迁移在旧 `odds_snapshots` 旁新增市场表与选项表；增强 `BusinessJsonMapper.mapOdds()` 把 JSON 赔率拆成“市场 + 选项”；新增 `oddscenter` 查询 API 与 `/odds` 前端页面。所有新增行为先写测试，遵守认证和只展示事实数据的边界。

**Tech Stack:** Java 17, Spring Boot 3.3.6, JdbcTemplate, Flyway, H2/MySQL, Vue 3, TypeScript, Element Plus, Vitest, OpenSpec。

# 核心数据模型 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 建立核心业务数据表，并把已批准 `import_items` 幂等映射进正式 MySQL 业务表，同时在前端提供轻量概览和正式入库入口。

**Architecture:** 后端新增 `coredata` 聚合，使用 Flyway 创建正式业务表，用 `JdbcTemplate + Jackson` 做弱结构 JSON 映射，复用 `ImportItemRepository` 读取 approved import item。前端新增 `coreData` API helper，Dashboard 展示概览，ImportReview 页面触发正式入库。

**Tech Stack:** Java 17, Spring Boot 3, Spring JDBC, Spring Data JPA, Flyway, Jackson, MockMvc, Vue 3, TypeScript, Element Plus, Vitest, OpenSpec.

private ImportItem saveItem(ImportItemType type, ImportItemStatus status, boolean validJson, String rawJson) {
    ImportJob job = new ImportJob();
    job.setArchivePath("test/archive");
    job.setStatus("SCANNED");
    job.setMessage("test");
    job.setTotalItems(1);
    job.setValidItems(validJson ? 1 : 0);
    job.setInvalidItems(validJson ? 0 : 1);
    ImportItem item = new ImportItem();
    item.setJob(job);
    item.setItemType(type);
    item.setStatus(status);
    item.setRelativePath(type.name().toLowerCase() + ".json");
    item.setSha256("0".repeat(64));
    item.setSummaryTitle("test");
    item.setValidJson(validJson);
    item.setValidationMessage(validJson ? "ok" : "invalid");
    item.setRawJson(rawJson);
    job.addItem(item);
    return jobRepository.save(job).getItems().get(0);
}

private int count(String table) {
    return jdbcTemplate.queryForObject("SELECT COUNT(*) FROM " + table, Integer.class);
}

- [ ] **Step 3: Add `MatchKeyNormalizer`**

Implement:

```java
public String normalize(String matchName, String matchday, String jcCode) {
    String base = Stream.of(matchName, matchday, jcCode)
            .filter(Objects::nonNull)
            .map(String::trim)
            .filter(s -> !s.isBlank())
            .collect(Collectors.joining("|"));
    return base.toLowerCase(Locale.ROOT).replaceAll("\\s+", " ");
}
```

- [ ] **Step 4: Add mapping/import services**

`CoreDataImportService.importItem(long itemId, String actor)` shall:

1. Load `ImportItem` from `ImportItemRepository`.
2. Reject unless `status == APPROVED` and `validJson == true`.
3. Return existing mappings if any mapping exists for the import item.
4. Parse raw JSON using Jackson.
5. Delegate to `BusinessJsonMapper`.
6. Insert audit log action `IMPORT_CORE_DATA`.

`BusinessJsonMapper` shall expose:

```java
List<CoreDataMappingResponse> map(ImportItem item, JsonNode json, String actor);
```

Internally route by `ImportItemType` and use `JdbcTemplate` for inserts/selects.

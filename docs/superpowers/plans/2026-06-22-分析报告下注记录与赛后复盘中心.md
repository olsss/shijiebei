# 分析报告下注记录与赛后复盘中心 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建分析报告、AI 下注方案、实际下注记录和赛后复盘中心，保存批准后的 JSON 并提供前后端查询展示。

**Architecture:** 使用 V7 迁移新增 AI 下注方案和复盘表，并扩展 `bets` 结算字段；增强 `BusinessJsonMapper` 从 ANALYSIS/BETS JSON 入库；新增 `analysisreviewcenter` 后端查询 API 与 `/analysis-review` 前端页面。所有新增行为先写测试并确认失败，再实现最小代码通过测试。

**Tech Stack:** Java 17, Spring Boot 3.3.6, JdbcTemplate, Flyway, H2/MySQL, Vue 3, TypeScript, Element Plus, Vitest, OpenSpec。

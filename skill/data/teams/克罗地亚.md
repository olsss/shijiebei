## 战意与排名形势（§2D）
- 当前出线情况：1 战 0 分（2-4 负英格兰），与巴拿马同为 0 分。本场对巴拿马是**生死战**：再负基本出局，平局也将使末轮极其被动。
- 名次诉求：必须争胜，无“选名次/划水”空间。
- 各名次对应的淘汰赛潜在对手 / 半区：尚不适用（先保出线）。
- 轮换倾向：生死战大概率主力尽出；但需权衡老将体能，Modrić/Kovačić 使用强度是看点。
- 默契球风险：低——双方都必须争胜。

## 大赛气质标签（§2A）
- 中场控制 / 拖节奏 / 大赛韧性强：传统强项，控球与经验突出。
- 风险面：本届首战暴露——抗快速反击差（“简单的过顶球”就被打穿）、定位球防守弱、老化导致下半场体能/节奏跟不上（被英格兰下半场连入 3 球）。
- 破密集能力：有 Modrić/Sučić/Baturina 的组织与远射（首战 2 球含一记世界波），面对巴拿马低位防守理论上有办法，但需防“控球占优却终结/效率不足”。
- 守领先能力：首战 45+5' 反超后下半场迅速崩盘，守领先能力本届打了问号。

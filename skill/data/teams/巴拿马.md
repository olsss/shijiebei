## 战意与排名形势（§2D）
- 当前出线情况：1 战 0 分（0-1 负加纳），积分垫底但同组克罗地亚同为 0 分；本场对克罗地亚是双方“事实上的出线生死战”，必须抢分。
- 名次诉求：尚非末轮，无“选名次”空间；需要尽量拿分以保留前二或最佳第三名希望。
- 各名次对应的淘汰赛潜在对手 / 半区：待末轮形势明朗。
- 轮换倾向：生死战大概率主力尽出，不会轮换。
- 默契球风险：低——双方都必须争胜，无双双小胜出线空间。

## 大赛气质标签（§2A）
- 低位防守 / 反击 / 定位球 / 守领先 / 拖节奏 / 身体对抗：身体对抗强、纪律性中北美风格；首战展示出可控球、可压迫，但守不住末段（90+5 被反击绝杀）。
- 破密集能力：有提升（控球 62%、502 次成功传球）但终结差（xG 0.75 / 12 射未进），破密集转化为进球能力存疑。
- 低比分韧性：首战 0 失球守到 90+5，低比分韧性尚可但末段崩盘风险高；0-0、0-1、1-0、1-1 路径要重点覆盖。

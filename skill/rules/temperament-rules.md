# 大赛气质 + 本届逐场表现 硬规则（CLAUDE.md §2A / §2B）

## §2A 世界杯/大赛气质硬规则
- 不能只因"赔率低、牌面强、战意强"就把球队当稳胆；必须核查该队在世界杯/大赛里的稳定性。
- 对"必须赢"的球队要反向审查：落后后是否急躁、面对低位防守是否有办法、是否容易无效传中/外围远射。
- 对南美、欧洲二线、非洲强队等身体和纪律性强的球队，要单独评估其低比分韧性，不能因首轮惨败就简单看衰。
- 若热门方缺少破密集能力，胜平负必须加厚平/负保险；比分必须覆盖 0-1、1-1、1-2 等低比分冷门路径。
- 红牌不自动等于优势方能进球；11打10后仍要看是否能形成高质量机会。
- 赛前结论必须写明：这是"实力判断""盘口判断"还是"大赛气质判断"，三者冲突时降低投入。

## §2B 本届世界杯逐场表现硬规则
- 下注前必须分析每支球队在本届世界杯本场之前的全部比赛表现；每支球队、每一场都要单独列出，不许偷懒。
- 每场已赛至少写清：对手、比分、进失球时间、首发/阵型变化、关键伤停/红黄牌、控球与射门质量、定位球、反击、防守稳定性、领先/落后后的应对。
- 不能只写"首轮赢了/输了/状态不错"；必须判断该场结果是实力体现、战术成功、对手失误、运气因素，还是比分掩盖问题。
- 若球队本届还没有已赛样本，必须明确写"本届首战，无本届样本"，再补预选赛、洲际杯、友谊赛和大赛历史。
- 当前赛事表现优先级高于旧数据；旧世界杯/历史交锋只能辅助，不能覆盖本届真实状态。
- 对比分下注尤其要据本届逐场修正：若球队多次低比分、慢热、终结差、守领先强，必须增加 0-0、0-1、1-1、1-0 等低比分路径。

## §2C 多源数据交叉验证硬规则（核验入口见 workflow/source-snapshot.md）
- 比赛数据不能只看一家；下注前至少使用 2 个以上独立来源交叉验证。
- 优先级：官方赛事/竞彩数据 > 权威数据商/比分平台 > 主流媒体战报 > 普通预测文章。
- 赛果、首发、伤停、红黄牌、进球时间、控球、射门、射正、角球、xG/机会质量等关键数据尽量多源核对。
- 来源冲突写明冲突点，按更权威或更接近官方者处理；不得选择性引用。
- 区分：数据事实 / 媒体观点 / 赔率信号 / 个人推断，不能混为事实。
- 时间紧无法完成多源核查 → 降低金额或放弃该场，不得用单一来源硬下结论。

## §2D 战意与排名操控硬规则（出线后 / 末轮）
- 已确保出线的球队，下注前必须判断其名次诉求：是否为淘汰赛对阵更有利而主动选第二/第三、甚至接受平局或小负，而非全力争头名。
- 末轮核对：出线形势、各名次对应的淘汰赛半区与潜在对手（是否避开强队/想换半区），判断真拼还是划水保留。
- 已出线球队可能大幅轮换、降强度、保主力避伤停黄牌停赛；“必须赢”与“无所谓”两端都要识别。
- 降低对纸面强队的胜负预期，增加平/小胜/冷门与低比分路径；牌面强 ≠ 有战意。结论标注“战意判断”，冲突时降投入。
- 警惕“默契球”：双方都只需小胜/平即可双双出线时的低强度比赛。
- 每队的出线情况、名次诉求、轮换倾向记录在 `data/teams/<队名>.md` 的“战意与排名形势”节。

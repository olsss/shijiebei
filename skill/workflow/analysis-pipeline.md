# 九维分析框架编排（§2，对应阶段 3–4）

> 每场至少跑完九个维度，再进证据加权（evidence-weight.md）和出票决策。

## 九维清单（逐项产出结论，不许跳）
1. 历史交锋与近期战绩
2. 阵容配置、主力状态、伤停/停赛（缺→置信度降级）
3. 教练状态、战术风格、轮换可能
4. 小组形势、战意、赛程体能、天气/场地（§2D：已出线队是否为淘汰赛对阵主动选名次/轮换/默契球）
5. 赔率与庄家视角：世界大公司赔率为主（Pinnacle 基准）+ 初赔→即时变化；体彩去水重仅取 SP，不作分析主依据
6. 世界杯/大赛履历（近几届世界杯/洲际杯/预选赛淘汰/小组赛）
7. 球队比赛气质（低位防守/反击/定位球/守领先/拖节奏/身体对抗）
8. 破密集能力（稳定中锋/边路爆点/倒三角/二点包抄/定位球质量；控球≠有效进攻）
9. 冷门路径（平局/小胜/卡让球线/末段失球/早丢球后急躁/11打10打不开）

## 编排顺序
1. 先取 `data/per-match/` 做 §2B 逐场分析（当前赛事 > 旧数据）。
2. 再补 `data/team-data.md` 历史档案（维度 1、6）。
3. 对热门方/必须赢的队执行 §2A 反向审查（落后是否急躁/能否破低位/无效传中远射）。
4. 汇入 evidence-weight.md 加权 + 置信度降级（含数据时效 §14、行为偏差 §13）。
5. 输出每场：主推方向 + **玩法选择（HAD/HHAD/比分/总进球/半全场，选方差最低赔率最优者，§12）** + 冷门路径清单 + 结论类型 + 置信度。
6. 淘汰赛场次先按 90 分钟口径表达判断（“晋级 ≠ 90 分钟赢”，§11）：预期加时点球解决 → 主推“平”/平局比分而非强队胜。
7. 末轮/出线相关场次先查 §15 最佳第三名战意（小组第 3 出线模糊 → 降胜负预期、加低比分/平局路径）。

## 红牌特别提醒
红牌不自动等于优势方进球；11 打 10 后仍要看能否形成高质量机会（§2A）。
